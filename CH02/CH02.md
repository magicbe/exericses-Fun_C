# CH02 - 變數、常數與資料型態

## 2-1 變數與常數

- 變數，表示會隨著程式的執行而改變其內容。
- 取變數名稱要注意下列幾件事：
    1. 第一個字一定要是英文字母英文字母，之後可接英文字母、數字和底線(_)
    2. 變數名稱可以讓大家很容易的看出它所代表的事項。
    3. 不可以取和保留字一樣的變數名稱。
    4. 變數名稱的大小寫是不相同。

## 2-2 資料型態

資料型態的用意在於配置適量的記憶體給變數，因為變數一定要放在記憶體才能運作。

資料型態|C語言表示法|佔記憶體的大小(bytes)
:----:|:----:|:----:
整數|int|4
浮點數|double|8

#### 輸出的格式特定字元
資料型態|格式特定字元
:----|:----
int|%d
double|%f

#### 輸入的格式特定字元
資料型態|格式特定字元
:----|:----
int|%d
double|%lf

## 2-3 關鍵字

- 變數 variable
- 常數 constant
- 資料型態 data type
- 整數 integer
- 浮點數 floating point
- 單準確度 single precision 浮點數
- 倍準確度 double precision 浮點數
- 格式特定字元 format specified character
